# Linux Day 3 - File Permissions and Process Management

## Objective

The objective of Day 3 was to understand Linux file permissions, ownership, users, groups, and basic process management.

---

## Topics Covered

- Linux file permissions
- Numeric permission values
- chmod command
- File ownership
- User and group information
- Running processes
- Process monitoring

---

## Commands Practiced

- chmod
- ls -l
- whoami
- id
- ps
- top

---

## Permission Values Learned

| Number | Permission |
|---------|------------|
| 7 | rwx |
| 6 | rw- |
| 5 | r-x |
| 4 | r-- |
| 0 | --- |

---

## Common Permission Examples

700 → rwx------

755 → rwxr-xr-x

644 → rw-r--r--

600 → rw-------

777 → rwxrwxrwx

---

## Learning Outcome

After completing Day 3, I can:

- Read Linux file permissions.
- Convert symbolic permissions into numeric values.
- Change file permissions using chmod.
- Understand owner, group, and others.
- View current user information.
- Monitor running Linux processes.
