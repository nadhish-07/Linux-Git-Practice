# Linux Day 1 Commands

## Show current directory

```bash
pwd
```

## List files

```bash
ls
```

## Detailed list

```bash
ls -l
```

## Create directory

```bash
mkdir practice
```

## Move into directory

```bash
cd practice
```

## Go back one directory

```bash
cd ..
```

## Create files

```bash
touch a.txt b.txt c.txt
```

## Write content into file

```bash
echo "Hello Linux" > a.txt
```

## Append content into file

```bash
echo "New line" >> a.txt
```

## Display file content

```bash
cat a.txt
```

## Copy file

```bash
cp a.txt d.txt
```

## Rename file

```bash
mv b.txt linux.txt
```

## Delete file

```bash
rm d.txt
```
