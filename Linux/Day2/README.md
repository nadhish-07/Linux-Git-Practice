# Linux Day 2 - File Content, Searching, Redirection and Filtering

## Objective

The goal of Day 2 was to learn how to work with file contents, search data, redirect command outputs, and filter information using Linux commands.

---

## Topics Covered

- Viewing file contents
- Creating content inside files
- Appending text to files
- Searching text using grep
- Counting lines using wc
- Redirecting output using > and >>
- Using pipes (|)
- Viewing file contents using cat
- Basic log analysis

---

## Commands Practiced

- cat
- echo
- grep
- wc
- >
- >>
- |

---

## Mini Project

Created log files containing INFO and ERROR messages.

Used grep command to filter ERROR logs.

Practiced viewing file contents and counting lines.

---

## Learning Outcome

After completing Day 2, I can:

- Create file contents using echo
- Search specific words from files
- Count lines, words and characters
- Redirect command output
- Append new content to files
- Perform basic log analysis
