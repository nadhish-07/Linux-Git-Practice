## Branch Workflow

```text
main
 │
 ├── feature-login
 │
 ├── feature-dashboard
 │
 └── feature-profile
```

Each feature is developed independently and merged into the main branch after completion.

---

## What is a Merge Conflict?

A merge conflict occurs when two branches modify the same line of the same file differently. Git cannot decide which change to keep automatically, so it asks the developer to resolve the conflict manually.
